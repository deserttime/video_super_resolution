# Build Guide

This project can be packaged as a Windows desktop application with PyInstaller.

## Recommended Environment

- Windows 10 or 11
- Python 3.11+
- FFmpeg available on the target machine, or bundled separately
- Optional: `realesrgan-ncnn-vulkan` distributed alongside the app

## Install Packaging Dependency

```powershell
python -m pip install pyinstaller
```

## Basic Build Command

```powershell
pyinstaller --noconfirm --windowed --name VideoSuperResolution --collect-all PySide6 video_sr\gui.py
```

## Build Output

After packaging, check:

```text
dist\VideoSuperResolution\
```

## Recommended Files To Ship Together

If you want the packaged app to work on another machine, ship these together:

- `VideoSuperResolution.exe`
- the required PySide6 runtime files generated by PyInstaller
- `ffmpeg.exe`
- `ffprobe.exe`
- optional `realesrgan-ncnn-vulkan.exe`
- optional `models\*.bin` and `models\*.param`

## Notes

- If FFmpeg is not in PATH, this project already tries the local fallback path used during development.
- If you want a cleaner release, consider copying FFmpeg and Real-ESRGAN into a dedicated `tools/` folder next to the packaged app and then updating the code to resolve relative paths.
- Test the packaged GUI on a clean machine before publishing a release.
